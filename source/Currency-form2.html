*Choose any money from "From currency" field
*Choose any money from "To currency" field
*Enter an amount at "Amount" field

Some extra informations:

This is a dynamic web app!

*The main use of this currency converter(which is to convert currencies) is implemented through a servlet class (CurrencySelect.class).

*The Swap Currencies button is implemented through a javascript function swap(a,b) where a is the value of "From Currency" field and b is the value of "To Currency field".

*The is implemented by another servlet class (DownloadCode.class) which class works through a "response.getOutputStream()"

*The has 2 options:

**Click option is implemented by a ServletContextListener through 3 classes:

1)MyServletContextListener which is the Listener class.

2)ReObj which is an attribute class

3)ListenServlet which is the servlet

**onMouseover option which is implemented through a css class (.borde) and a javascrip function showborder(id) where the id is defined into a info tag as the class css class has info.borde.
